Découvrez comment appliquer des filtres CSS à vos images
pour les rendre plus attrayantes et dynamiques.

## blur

La fonction CSS **blur()** permet d'appliquer un flou gaussien sur l'image d'entrée.
Le résultat de cette fonction est une valeur **filter-function**.

```css
img {
    filter: blur(5px);
}
```

Un petit exemple visuel de la fonction **blur()** :
<Filters image={cat} data={[
    { filter: "none", label: "Originale" },
    { filter: "blur(2px)", label: "2px" },
    { filter: "blur(5px)", label: "5px" }
]} />

## brightness

La fonction CSS **brightness()** applique un multiplicateur linéaire sur une image
afin de la rendre plus claire ou plus sombre.
Le résultat obtenu est une valeur de type **filter-function**.

```css

    filter: brightness(50%);
}
```

Un petit exemple visuel de la fonction **brightness()** :
<Filters image={cat} data={[
    { filter: "none", label: "Originale" },
    { filter: "brightness(50%)", label: "50%" },

]} />
